# 2048
Final Project 2025 [2048]
----------------- HOW TO PLAY -----------------

RUNNING THE GAME
1. Make sure Python is installed.
2. Run the game using:
   python main.py
3. The game window will open.
4. Click "Start Game" to begin.

CONTROLS
- Move Up    : Up Arrow key
- Move Down  : Down Arrow key
- Move Left  : Left Arrow key
- Move Right : Right Arrow key

GAME RULES
- Tiles with the same number will merge into a higher number.
- Every merge increases your score.
- A new tile appears after each valid move.
- The game ends when no more moves are available.
